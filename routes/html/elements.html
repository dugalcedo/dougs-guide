example this is... more specifically... also called... What does it do?
<p id="abc" class="xyz">Welcome to my website</p> an element a "p" or "paragraph" element --- It's like a little container for some content on a webpage
<p> a tag a start tag an opening tag It marks the beginning of an element
</p> a tag an end tag a closing tag It marks the end of an element
p a tagname p --- Determines the type of element.
id="abc" an attribute an id attribute --- Attributes can do many things, but this particular one assigns a unique identifier to an element, making it easier to refer to later.
Welcome to my website content text content --- It is what the element contains. This can be plain text, or even more elements.
<img src="moose.jpg" alt="Moose">
<img src="moose.jpg" alt="Moose" /> 		an element an empty element a self-closing element It contains no content, and therefore has no end tag. NOTE: you may see them written with a "/" before the final angle bracket. This is not necessary in normal HTML, but just know that you may come across them.
