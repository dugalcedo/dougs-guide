CSS Combinators

Combining selectors

combinator name what it does example What the example would select.
[space] descendant combinator Selects elements within another element. #main-navbar li { padding: 10px; } Any element with a tagname of "li" that is inside of the element with an id of "main-navbar"
> child combinator Selects elements that are a direct child of another element. In other words, it's like the descendant combinator, but only looks one level down. main > * { width: 1200px; max-width: 100%; margin: auto; } Any element of any tagname that is a direct child of elements that have a tagname of "main".
+ first sibling combinator Selects elements that are the first sibling after another element. input + button { margin-left: 1rem; } Any button that comes directly after an input.
~ general sibling combinator Selects all sibling elements that come after another element. In other words, it's like the first sibling combinator, but selects all siblings after. Not just the first. .gallery-photo ~ blockquote { font-style: italic; } Any element with a tagname of "blockquote" who is a sibling that comes after an element with a class of "gallery-photo".
