Higher-order array methods

method what it does parameters callback parameters what the callback should return
Array.forEach(cb) Executes a callback for each element in the array.
  • cb: callback(x, i, a)
  • x: array element
  • i: iteration count
  • a: original array
 nothing
example
const numbers = [2, 4, 6, 8, 10] numbers.forEach((n, i) => { console.log(n*i) }) Console:
0 4 12 24 40
Array.map(cb) From the array, returns a new array of the same length based on what the provided callback returns. same as above The desired array element. If nothing is returned, undefined will be pushed to the new array.
example
const animals = [ {name: 'moose', order: 'mammalia'}, {name: 'eagle', order: 'aves'}, {name: 'seal', order: 'mammalia'} ] const animalNames = animals.map((animal) => { return animal.name }) console.log(animalNames) Console:
['moose', 'eagle', 'seal']
Array.filter(cb) From the array, returns a new array containing only the original elements where the provided callback returned something truthy. same as above A value which is truthy if the current element is desired in the new array.
example
same as above const mammals = animals.map((animal) => { return animal.order === 'mammalia' }) console.log(mammals) Console:
[ {name: 'moose', order: 'mammalia'}, {name: 'seal', order: 'mammalia'} ]
Array.find(cb) Same as fiter, but instead of returning an array, it only returns the first match. If none are found, it returns undefined. same as above
example
same as above const eagle = animals.find((animal) => { return animal.name === 'eagle' }) console.log(eagle) Console:
{name: 'eagle', order: 'aves'}
Array.every(cb) Returns true if the provided callback always returns something truthy, else false. same as above A value which is truthy or falsy, representing some condition which every array element must meet.
example
same as above const allAnimalsAreMammals = animals.every((animal) => { return animal.order === 'mammalia' }) console.log(allAnimalsAreMammals) Console:
false
